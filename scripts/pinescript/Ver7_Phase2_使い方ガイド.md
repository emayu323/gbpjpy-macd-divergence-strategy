# GBPJPY Ver7 Phase2 - エントリーシグナルインジケーター 使い方ガイド

## 📊 概要

**Ver7_Phase2_Entry.pine** は、GBPJPY System Ver7のエントリータイミング判定部分を独立させたインジケーターです。

### 主な機能

1. **RCI短期(9)・中期(14)計算**
2. **RCI反転シグナル検出**
3. **エントリータイミング矢印表示**
4. **リアルタイムRCI値表示**

---

## 🚀 TradingViewへの導入手順

### 1. インジケーター追加

1. TradingViewで **GBPJPY 5分足チャート** を開く
2. 「インジケーター」をクリック
3. 「Pine エディタ」タブを開く
4. `Ver7_Phase2_Entry.pine` の内容をコピー&ペースト
5. 「保存」→「チャートに追加」

### 2. 表示位置

- **メインチャート**（ローソク足と同じエリア）に表示されます
- エントリーシグナルを矢印で表示

### 3. 推奨設定

デフォルト設定で問題ありませんが、調整可能な項目：

| 設定項目 | デフォルト | 説明 |
|---------|-----------|------|
| RCI短期期間 | 9 | 短期RCI計算期間 |
| RCI中期期間 | 14 | 中期RCI計算期間 |
| RCI短期閾値 | 60 | 短期RCI反転判定閾値（±60） |
| RCI中期閾値 | 40 | 中期RCI確認閾値（±40） |
| シグナル矢印表示 | ON | エントリー矢印表示/非表示 |
| 情報パネル表示 | ON | 左上パネル表示/非表示 |

## 📈 チャートの見方

### エントリーシグナル矢印

| シグナル | 色 | 位置 | 意味 |
|---------|---|------|------|
| **▲ 買** | ライム（緑） | ローソク足下 | 買いエントリータイミング |
| **▼ 売** | フクシア（紫） | ローソク足上 | 売りエントリータイミング |

### 情報パネル（左上）

リアルタイムで以下を表示：

| 項目 | 表示内容 |
|------|---------|
| **RCI短期(9)** | 現在のRCI短期値 |
| **RCI中期(14)** | 現在のRCI中期値 |
| **短期方向** | RCI短期の方向（↑/↓/→） |
| **シグナル** | 買い/売り/待機 |

#### パネル色の意味

**RCI値の背景色**:
- 青系: 売られすぎゾーン（-60以下/-40以下）
- 赤系: 買われすぎゾーン（+60以上/+40以上）
- 灰色: 中立ゾーン

**シグナルの背景色**:
- ライム: 買いシグナル発生
- フクシア: 売りシグナル発生
- 灰色: 待機中

## 🎯 実践的な使い方

### ✅ 買いシグナルの条件

1. **RCI短期(9)** ≤ -60（売られすぎゾーン）
2. **RCI短期(9)** が反転上昇（Hook Up）
3. **RCI中期(14)** ≤ -40 で上昇中

→ **緑の▲矢印**が表示されたら買いエントリータイミング

### ❌ 売りシグナルの条件

1. **RCI短期(9)** ≥ +60（買われすぎゾーン）
2. **RCI短期(9)** が反転下落（Hook Down）
3. **RCI中期(14)** ≥ +40 で下落中

→ **紫の▼矢印**が表示されたら売りエントリータイミング

### 📌 シグナルの使い方

#### Phase2単独で使用する場合
- 矢印が出たらエントリー検討
- ただし、**環境認識なし**なので勝率は低い可能性あり

#### Phase1と組み合わせる場合（推奨）
1. **Phase1**: 背景がライム/フクシアになる（エントリー可能状態）
2. **Phase2**: 矢印が出る（エントリータイミング）
3. **両方揃った時のみエントリー**

→ Phase3（統合版）でこれを自動化します

## 🔔 アラート設定

### 推奨アラート

1. **買いシグナル発生**
   - 条件: RCI短期反転上昇+中期確認
   - 使用タイミング: 買いエントリー検討

2. **売りシグナル発生**
   - 条件: RCI短期反転下落+中期確認
   - 使用タイミング: 売りエントリー検討

### アラート設定手順

1. インジケーター名の右にある「…」をクリック
2. 「アラートを追加」を選択
3. 条件を選択（「買いシグナル発生」または「売りシグナル発生」）
4. 通知方法を設定（アプリ通知、メール、Webhook等）
5. 「作成」

## 🔍 トラブルシューティング

### Q1: シグナルが出ない

**原因**:
- RCIが閾値に到達していない
- 反転条件を満たしていない
- RCI中期が動いていない

**解決策**:
- 情報パネルでRCI値を確認
- RCI閾値を調整（60→50、40→30など）
- より多くのシグナルを見たい場合は閾値を下げる

### Q2: シグナルが多すぎる

**原因**:
- RCI閾値が低すぎる
- 環境認識なしで使用している

**解決策**:
- RCI閾値を上げる（60→70、40→50など）
- **Phase1と組み合わせて使用**（推奨）

### Q3: 情報パネルが邪魔

**調整方法**:
- 「設定」→「表示設定」
- 「情報パネル表示」をOFF
- シグナル矢印だけ表示される

## 📝 重要な注意事項

### ⚠️ このインジケーターだけではトレードしない

Phase2は **エントリータイミング判定のみ** です。実際のエントリーには：

1. ✅ **Phase1**: 環境認識（トレンド+ダイバージェンス）
2. ✅ **Phase2**: エントリータイミング（RCI反転シグナル）
3. ⏳ **Phase3**: 統合版（Phase1 + Phase2 + SL/TP）

**Phase1とPhase2の両方の条件が揃って初めてエントリー**します。

### 📊 推奨チャート構成

```
┌─────────────────────────────┐
│ GBPJPY 5分足（メインチャート） │
│                             │
│ ・価格アクション             │
│ ・Phase1環境認識（背景色）    │
│ ・Phase2エントリー矢印        │
│ ・情報パネル（左上）         │
└─────────────────────────────┘
```

**Phase1とPhase2を同時に表示**することで、環境認識とエントリータイミングを両方確認できます。

## 🎓 学習のヒント

### RCIの理解

1. **RCI短期(9)**
   - 短期的な価格の勢い
   - 反転を早く捉える
   - ダマシも多い

2. **RCI中期(14)**
   - 中期的なトレンド確認
   - シグナルの信頼性を高める
   - 短期と組み合わせることで精度向上

### 練習方法

1. **過去チャートで検証**
   - リプレイ機能を使用
   - シグナルの精度を確認
   - Phase1背景色とシグナル矢印の関係を観察

2. **リアルタイム観察**
   - 1-2週間観察
   - シグナルの出現頻度を把握
   - Phase1と組み合わせた時の勝率を確認
